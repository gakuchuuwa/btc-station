# BTC Station: 极客量化工作台 (PRD)

> **版本定位**：v4.0 (100% Free / Feature-First Edition)
> **核心原则**：只做 BTC。极速体验。完全免费。商业化代码（付费墙/配额拦截）一律禁止引入。

## 一、 产品定位 (What & Why)
这是一个专为中高阶 BTC 交易员打造的**Web版专业量化投研平台**。
解决的核心痛点：
1. TradingView 测算复杂策略（尤其含止盈止损、资金费率）极其不准。
2. TradingView 的 Assistant 插件跑穷举调参极慢且吃内存。
3. 交易员需要一个能无缝衔接 `quant-lab.org` (稳健性分析) 的一条龙工作台。

## 二、 核心功能模块 (Features)

### 1. BTC 全景仪表盘 (Homepage)
*   提供最全面的 BTC 实时行情、多周期图表与链上/宏观新闻流。
*   降低用户防备心，作为纯粹的看盘工具也极具价值。

### 2. 策略回测实验室 (Strategy Editor)
*   **交互**：类似 TradingView 的浏览器内看盘与代码编辑体验 (Monaco Editor)。
*   **引擎**：基于 Python + Freqtrade，支持严谨的永续合约回测逻辑。
*   **速度**：后端预载历史数据，免去 API 抓取等待。

### 3. Hyperopt 智能调参集群 (Optimizer)
*   **能力**：彻底取代本地电脑或浏览器插件跑回测。
*   **算法**：支持贝叶斯优化、退火法等启发式算法，寻找收益/回撤的最佳参数组合。
*   **展示**：2D/3D 参数高原区可视化面板。

### 4. Quant-Lab 标准导出 (Data Exporter)
*   **规范**：每次回测/调参必须能一键导出 **191 列 TV 兼容标准 CSV**。
*   **对接**：保证导出的 CSV 能够直接喂给 `https://quant-lab.org/` 进行过拟合检验与蒙特卡洛稳健性分析。

### 5. 模拟/实盘常驻引擎 (Live Daemon)
*   基于 Freqtrade 的 `trade` 模式。
*   一键把回测通过的代码挂上服务器，进入无人值守的实盘/模拟盘执行状态。
